# Contributing to Carbon

First off, thank you for considering contributing to Carbon! It's people like you who make it a great tool for the community.

## 💬 Have a Question?

If you have questions about how to use Carbon or need help with setup, please **join our Discord server**. It's the fastest way to get support and connect with other users and developers.

[**Join our Discord Server**](https://discord.gg/y5HRSfATwA)

## 🐛 Found a Bug?

If you've encountered a bug, we'd love to hear about it! Before creating an issue, please:

1.  Check the [existing issues](https://github.com/sudoscrawl/carbon/issues) to see if it has already been reported.
2.  If not, **create a new issue**. Be sure to include:
    - A clear and descriptive title.
    - Steps to reproduce the bug.
    - What you expected to happen vs. what actually happened.
    - Any relevant logs or screenshots.

## 💡 Feature Requests

Have an idea for a new feature? We're open to suggestions! Please create an issue and use the "Feature Request" label. Describe the feature in detail and why it would be beneficial for Carbon users.

## 🛠️ Submitting a Pull Request (PR)

We welcome code contributions! To ensure a smooth review process, please follow these steps:

1.  **Fork the Repository:** Create your own fork of the Carbon repository.
2.  **Create a Branch:** Create a branch for your changes (e.g., `feature/new-moderation-command` or `fix/i18n-bug`).
3.  **Make Your Changes:** Implement your feature or fix.
4.  **Follow Coding Standards:**
    - Ensure your code follows the existing style and conventions.
    - Run `ruff check .` and `ruff format .` to ensure linting and formatting pass.
    - Run `pyright` to check for type errors.
5.  **Add Tests:** If you're adding a new feature, please include relevant tests.
6.  **Submit the PR:** Push your branch to your fork and create a Pull Request against the `main` branch of the original repository.
7.  **Describe Your PR:** Provide a clear description of what your PR does and link it to any relevant issues.

## 📜 Code of Conduct

By participating in this project, you agree to abide by our [Code of Conduct](.github/CODE_OF_CONDUCT.md).

Thank you for your contributions!
